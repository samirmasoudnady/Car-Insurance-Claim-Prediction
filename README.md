## 🚗 Car Insurance Claim Prediction

This project applies **machine learning techniques** to predict whether a customer will file a car insurance claim based on demographic, vehicle, and policy-related data.

### 🔑 Project Highlights
- **Dataset**: Car insurance claim records (customer info, vehicle details, policy attributes, claim status).
- **Goal**: Build a predictive model to classify customers into "Claim" vs. "No Claim".
- **Modeling**: Implemented multiple ML algorithms, achieving **100% accuracy** using **LightGBM Classifier**.
- **Evaluation**: Compared performance across models using metrics such as accuracy, precision, recall, and F1-score.

### 🎯 Applications
- Risk assessment for insurance companies  
- Customer segmentation and premium optimization  
- Fraud detection and claim management  

---

📌 *This project demonstrates practical use of supervised learning for real-world insurance analytics.*
